# Slide Template

## Slide outline

Every fivebucks.ai campaign deck follows this 10-slide master structure. Adjust slide count by expanding or compressing sections — never skip the arc.

### Standard 10-Slide Structure

| Slide | Title | Purpose |
|-------|-------|---------|
| 1 | Cover | Campaign name, persona, date |
| 2 | The Market Context | What's happening in the persona's world right now |
| 3 | The Problem | Specific pain point this campaign addresses |
| 4 | Why Now | The market gap / urgency / why fivebucks.ai is positioned to win |
| 5 | The Campaign Concept | What we're doing and the core message |
| 6 | How It Works | Key fivebucks.ai features at the center of the campaign |
| 7 | The Plan | Channels, content types, timeline |
| 8 | Expected Results | KPI targets and success metrics |
| 9 | Budget & Resources | What's needed to execute |
| 10 | Next Steps / Ask | Decision required, owner, timeline |

### Compressed 6-Slide Structure (executive audience)

| Slide | Title | Purpose |
|-------|-------|---------|
| 1 | Cover | Campaign name, date |
| 2 | Situation & Opportunity | Context + problem + why now (combined) |
| 3 | The Campaign | Concept + key messages + fivebucks.ai features |
| 4 | Plan & Timeline | Channels, deliverables, schedule |
| 5 | Expected Results | KPI targets |
| 6 | Ask | Decision, owner, next steps |

## Copy rules

### Slide titles
- **Maximum 10 words** — if it's longer, it's a paragraph, not a title
- Write titles as **statements**, not labels:
  - ❌ "Market Overview" → ✅ "SEO professionals are losing traffic to AI search right now"
  - ❌ "Campaign Plan" → ✅ "3-channel campaign targeting Marcus across LinkedIn, email, and search"
- The title = the one thing the audience should remember from this slide
- No jargon in titles — if a non-marketer can't understand it, rewrite it

### Slide body copy
- **Maximum 5 bullets per slide** — if you have more, split the slide
- Each bullet: **maximum 12 words**
- Bullets should be verb-led where possible:
  - ❌ "AI Search optimization capability" → ✅ "Tracks ChatGPT, Claude, and Perplexity citations in one dashboard"
- No full sentences in bullets — phrases only
- No period at the end of bullets
- **One data point per slide maximum** — don't data-dump

### Brand voice rules
- Confident but not hypey — no "game-changing" or "revolutionary" without specific proof
- Use approved phrases from `brands/{brand}/brand.md` where they fit: "Optimize for Google & AI Search", "Traffic generation + lead generation in one place"
- No guaranteed ranking promises or timeframes (except the 90-day managed service guarantee — cite exactly)
- Avoid passive voice — use active, verb-forward language

### Numbers and data
- Always cite the source when using a number (e.g., "275M+ contacts (fivebucks.ai database)" or "25% traffic increase — 90-day managed service guarantee")
- Round numbers to clean figures in slides (say "275M+" not "275,432,891")
- Use % changes for comparisons: "+28% traffic vs. prior month"
- Do not invent metrics — only use data from context files or user-provided data

## Chart rules

### When to use charts
- Use charts when a visual representation communicates the trend faster than text
- Maximum 1 chart per slide
- Don't use charts for 2-3 data points — use a table or bullet instead

### Approved chart types

| Chart Type | Use For | Avoid When |
|-----------|---------|------------|
| Bar chart | Comparing categories (competitor pricing, plan features) | More than 6 categories |
| Line chart | Showing trends over time (traffic growth, ranking improvement) | Less than 3 time periods |
| Funnel chart | Showing the marketing/sales funnel (traffic → leads → replies) | Non-sequential data |
| Comparison table | Feature/pricing comparisons (fivebucks.ai vs. competitor) | More than 5 columns |
| Donut/pie chart | Showing proportions (traffic source breakdown) | More than 4 segments |

### Chart design rules
- **Chart title**: Write as a statement, not a label — "Traffic grew 18% after 2 months of AI-optimized content" not "Traffic Chart"
- **Colors**: Use `#6b21a8` (purple) for fivebucks.ai bars/lines, `#e5e7eb` (gray) for competitors or prior periods
- **Labels**: Always label data points on bar charts — don't make the audience read the axis
- **Legend**: Only include a legend if there are 2+ series — never for a single-series chart

### Comparison table format (fivebucks.ai vs. competitor)

```
| Feature | fivebucks.ai | [Competitor] |
|---------|-----------|--------------|
| Feature name | ✅ Description | ❌ Not available |
| Feature name | ✅ Description | ⚠️ Partial (limitation) |
| Monthly cost | $39-$350 | $100-$450 |
```

Use ✅ for included, ⚠️ for partial/limited, ❌ for not available.

## Call-to-action slide template

The final slide (Ask / Next Steps) always uses this structure:

### Standard Ask slide

```markdown
## Slide [N]: [Clear ask in one line]

**What we need:**
- [Decision 1]: [e.g., Approve campaign budget of $X]
- [Decision 2]: [e.g., Greenlight content production starting [date]]
- [Decision 3]: [e.g., Assign campaign owner]

**Timeline:**
- [Date]: [Milestone 1]
- [Date]: [Milestone 2]
- [Date]: [Milestone 3 — e.g., Campaign goes live]

**Owner:** [Name or role responsible]

**Questions?** [Contact or reply channel]
```

### Client-facing proposal CTA slide

```markdown
## Slide [N]: Ready to move forward?

**Next step:** Start your 14-day free trial — no credit card required.

Or book a 20-minute demo to see [specific feature relevant to their use case].

**What happens next:**
1. [Action 1 — immediate]
2. [Action 2 — within 24 hours]
3. [Action 3 — e.g., First campaign live within a week]

[Start free trial →] [Book a demo →]
```

### Visual note for CTA slide
- Use the purple-to-pink gradient background (`linear-gradient(135deg, #6b21a8 0%, #ec4899 100%)`)
- White text throughout
- CTA buttons: white background with purple text (inverted from standard)
- Keep it visually clean — this slide should have the least text of any slide in the deck
